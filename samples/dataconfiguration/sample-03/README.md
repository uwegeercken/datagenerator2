## Sample 03

The sample data configuration file in this folder can be used to generate random test data.

### Structure
Following data is generated:
- an automatically generated unique row number
- a url column using a regular expression pattern
- an identifier column using a regular expression pattern


### Sample Output in Json Format

    {
        "autonumber":0,
        "url":"www.AaAGpQGBZoxVT.com",
        "identifier":"UhSugCssTWIWH8562"
    }
